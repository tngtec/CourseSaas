# CourseSaas: Sistema de Gestión Estudiantil

Sistema desarrollado para la gestión integral de alumnos y cursos, enfocado en facilitar el registro, control y seguimiento académico.

## 🚀 Tecnologías Utilizadas
* **Lenguaje:** PHP
* **Base de Datos:** MySQL
* **Frontend:** HTML5, CSS3, JavaScript, Bootstrap
* **Entorno de desarrollo:** XAMPP

## 📋 Funcionalidades Principales
* **Autenticación:** Sistema de Login y Registro seguro para usuarios.
* **Gestión de Alumnos:** CRUD completo para gestionar la información de los estudiantes.
* **Filtros Dinámicos:** Búsqueda y filtrado de alumnos por curso para una mejor organización.
* **Dashboard:** Panel central para la administración del sistema.

## 🛠 Instalación
1. Clonar el repositorio.
2. Importar la base de datos en tu gestor MySQL (XAMPP).
3. Configurar la conexión en el archivo `conexion.php`.
4. Ejecutar el proyecto en tu servidor local.
